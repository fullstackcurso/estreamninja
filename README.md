# EStreamNinja Repository

Repositorio oficial del addon EStreamNinja para Kodi. Permite instalar y actualizar automáticamente el addon desde Kodi.

## Instalación

### Opción 1: Desde el Administrador de Archivos de Kodi (Recomendado)
1. Abre Kodi y ve a **Ajustes > Administrador de archivos > Añadir fuente**
2. Introduce la URL:
```
https://fullstackcurso.github.io/estreamninja/
```
3. Ponle un nombre (por ejemplo, `StreamNinja Repo`)
4. Ve a **Ajustes > Addons > Instalar desde archivo ZIP**
5. Selecciona la fuente **StreamNinja Repo** y elige `repository.streamninja-1.0.0.zip`
6. Espera la notificación de instalación
7. Ve a **Instalar desde repositorio > StreamNinja Repository > Addons de vídeo > StreamNinja**

### Opción 2: Instalación manual mediante archivo ZIP
1. Descarga `repository.streamninja-1.0.0.zip` desde este repositorio
2. En Kodi, ve a **Ajustes > Addons > Instalar desde archivo ZIP**
3. Navega hasta el archivo descargado y selecciónalo
